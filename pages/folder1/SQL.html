# 介绍&安装

这是一篇数据库入门教程，

让你花费最少的精力就可以走完从创建数据库-->创建数据表-->表里添加数据-->查看表里参数-->查看表里参数存储信息，

一条龙服务，你还不心动？

嗯？

先来介绍下数据库安装：

这里我们用开源的MySQL来示范：

首先登陆官网【入口 (opens new window)】

选择Windows (x86, 32-bit), MSI Installer-----离线安装

下载好安装包后，双击后缀名为msi的安装文件安装

  1. 第一项我们选择Custom自定义安装
  2. 从左边列表里选择第一项MySQL下的MySQL Server 8.0下的MySQL Server 8.0.27 - X64，点击选中后再点击中间的第一个向右的箭头添加到左边，再选择MySQL Connector--> Connector/ODBC->Connector/ODBC 8.0 【选择与自己装的MySQL同版本】，之后点击Next下一步
  3. 跳转到只有你选择的那两个东西后选择Execute进行安装
  4. 安装完成点击Next进行配置MySQL
  5. 点击Next继续
  6. Config Type处选中Development Computer，下面Connectivity勾选TCP/IP，port填写3306，X Protocol Port填写33060，勾选Open Windows Firwall ports for network access，其余都不勾选/不填写，点击Next继续
  7. 保持默认选择的黄色感叹号第一项，Next继续
  8. 配置密码，重复输入你第一段填写的密码，Next继续
  9. 保持默认，Service Name保持默认MySQL80即可，看下下面的单选原点选择的是不是第一项，是的话Next继续
  10. 点击execute执行，直至完成，此时，MySQL已经配置完成，点击Finish
  11. 继续Next，Finish结束

安装成功后，验证

通过开始菜单（Win键）调用MySQL命令行客户端（MySQL Command Line Client），就可以进行MySQL操作了。
